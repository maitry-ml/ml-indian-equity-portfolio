# ML-Based Indian Equity Portfolio

This project builds a predictive system for 6 Indian stocks:
- RELIANCE
- HDFCBANK
- INFY
- M&M
- BHARTIARTL
- HUL

## Features
- Market data (OHLCV)
- Quarterly fundamentals
- Macro indicators
- Market sentiment proxies

## Model
- Walk-forward validation
- Regularized regression / XGBoost
- Rank-based long-short portfolio

## Performance Metrics
- Sharpe Ratio
- Max Drawdown
- Hit Ratio
- Equity Curve
